# Project 1 – Todo App (Frontend)

## Giới thiệu
Đây là phần frontend của ứng dụng Todo App, được xây dựng bằng React.
Ứng dụng cho phép người dùng quản lý danh sách công việc gồm:
- Xem danh sách công việc
- Thêm công việc mới
- Xóa công việc

Frontend giao tiếp với backend thông qua REST API.

---

## Công nghệ sử dụng
- React (Create React App)
- HTML, CSS, JavaScript
- Fetch API
- Git & GitHub
- CI/CD với Vercel
